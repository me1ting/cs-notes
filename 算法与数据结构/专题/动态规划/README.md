# 什么是动态规划？

`动态规划`是`dynamic programming`（简称`DP`）的翻译，无论是译名还是原名都与其算法原理没有直接联系，也难以考证当初为何取这个名字。

>动态规划通过组合子问题的解来求解原问题，应用于子问题存在重叠的情况。
>
>by 《算法导论》

# 状态和转移方程

术语`状态`用来描述子问题的答案。

术语`转移方程`用来描述在动态规划问题中，`状态`之间的关系，如菲波那锲数：

```
f(x) = f(x-1)+f(x-2)
```

# 滚动数组

`滚动数组`指在动态规划的实现中，使用一个数组记录部分状态而非所有状态，适合`求斐波那契数`这样的问题。
